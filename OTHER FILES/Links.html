Abilities
Races We can ignore favored classes for now.
Classes For now, ignore multiclass, prestige classes, and NPC classes.
Feats
Skills We want to keep in mind the following: does this skill have an armor penalty? Does it require training (at least one skill point) to use? Some feats will buff skills, and having 5 ranks in some skills will buff other skills.
Magic Items This may be a version 2 feature. Some of these are modifiers for existing armor or weapons, could lead to a pain trying to account for the combinations.
Armor "Maximum dexterity bonus" will need to be factored in to skills that use Dexterity and Armor Class.
Weapons There's a nice table halfway down this page.
Goods and services, mundane items A lot of these can give bonuses to skills in some situations. For now we just want to be able to show if a user has the item in inventory or not.
